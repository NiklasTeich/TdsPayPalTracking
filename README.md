# TdsPayPalTracking

Shopware 6 plugin to send PayPal tracking data after receiving MerWare shipping information

## System requirements

* Shopware >= 6.3
* simplexml
* dom
* fileinfo
* json

## Installation

* install plugin
* activate plugin
* clear caches

## Features

* send PayPal tracking data
* tracking number
* carrier (GLS/other)
